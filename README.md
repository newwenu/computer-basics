# 计算机组成原理学习笔记

## 项目说明

这是个人整理的计算机组成原理学习资料，主要包含基础概念的解释和技术细节的整理。

## 当前文档

目前版本控制中包含：

- **计算机组成基础/计算机组成原理（基础版）.md** - 基础概念整理
- **计算机组成基础/计算机组成原理（详细版）.md** - 技术细节文档

## 内容特点

- 尝试用生活化的比喻解释技术概念
- 整理了一些硬件参数和性能数据作为参考
- 从基础到技术细节的渐进式整理

## 涵盖内容

主要包括：

- 计算机基本硬件组件的工作原理
- 存储系统的层次结构
- 性能优化的基本方法
- 并行处理的基本概念
